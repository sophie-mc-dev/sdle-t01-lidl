# feup-sdle

### There are two ways to run the project:

## 1

1.1 - run server in sdle_t01_lidl directory:

    python -m sdle_t01_lidl.server.server

1.2 - run client in sdle_t01_lidl directory:

    python -m sdle_t01_lidl.client.client
